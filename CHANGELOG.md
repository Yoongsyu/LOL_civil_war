## 기능 개선

### 📊 플레이어 상세 정보
- 포지션별 승률 / 챔피언별 승률 라디오 탭으로 전환 가능
- 챔피언별 승률: `matches.json` 기록 기반 동적 집계
  - 챔피언 초상화 아이콘 표시
  - 승률 색상 구분 (60% 이상 초록 / 40% 미만 빨강)
  - 판수 내림차순 정렬
  - 4~5줄 고정 높이 스크롤 박스

### ✏️ 경기 기록 수정 기능
- 경기 기록 조회 화면에 **수정** 버튼 추가
- 수정 가능 항목: 승리팀, 포지션, 챔피언, 밴 목록
- 저장 시 기존 전적(stats) 원복 후 새 데이터로 재적용
- 취소 버튼으로 수정 폼 닫기

### 🚫 밴픽 등록
- 전적 입력 폼에 블루/레드 각 5밴 선택 섹션 추가
- 경기 레코드에 `bans` 필드로 저장
- 경기 기록 조회 시 밴 목록 표시

### 📋 전적 입력 UX
- 포지션 기본값을 탑→정글→미드→원딜→서포터 순으로 자동 설정

## UI / 디자인 개선

### 🃏 팀 구성 결과 카드
- 텍스트 목록 → 카드 형태로 변경
- 블루팀(파란 테두리) / 레드팀(빨간 테두리) 헤더 카드
- MMR 차이 색상 구분 (100 이하 초록 / 250 이하 노랑 / 초과 빨강)
- 각 플레이어 카드에 모스트 챔피언 초상화(원형, 44px) 표시
  - 포지션 고정 팀 구성 → 해당 포지션 모스트 챔피언
  - 랜덤 팀 구성 → 전체 통합 모스트 챔피언

### 🎨 티어 뱃지 시각화
- 텍스트 + 이모지 → 색상 배지로 교체
- 티어별 고유 배경/텍스트 색상
  - Iron 회색 / Bronze 갈색 / Silver 슬레이트 / Gold 황금
  - Platinum 청록 / Emerald 초록 / Diamond 파랑
  - Master 보라 / Grandmaster 빨강 / Challenger 금색
- 적용 위치: 플레이어 목록, 팀 구성 결과, 플레이어 상세 정보

### 📱 모바일 반응형
- 768px 이하: 컬럼 자동 줄바꿈, 폰트·여백 최소화
- 480px 이하: 전체 컬럼 100% 너비, 탭 레이블 압축

### ☑ 전체선택 UI 변경
- "전체선택" / "전체해제" 버튼 2개 → 헤더 행 체크박스 1개로 통합

### 📐 리더보드 정렬 UI
- 정렬 기준 / 최소 경기 수 입력란 레이블 높이 통일 (나란히 정렬)
